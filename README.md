# CV 静态简历网页

一个使用原生 `HTML/CSS/JS` 实现的静态简历网页项目，采用 **A4 纸张视觉布局**，支持**在线编辑、数据驱动渲染、自动分页、PDF 导出**。

## 项目特点

- **数据驱动渲染** — 简历内容由 `data.json` 管理，修改数据即可更新页面
- **内置编辑器** — 页面内直接编辑简历所有模块，所见即所得
- **多种导入/导出** — 支持 **JSON** 和 **Markdown** 格式的导入与导出
- **自动分页** — 内容超出单页时自动拆分，多页显示页码
- **PDF 导出** — 一键导出为 PDF 文件
- **本地存储** — 编辑内容自动保存到浏览器 `localStorage`
- **时间轴功能** — 可视化呈现职业生涯时间线
- **头像上传** — 支持上传和预览头像图片
- **缩放控制** — 右下角悬浮工具栏，范围 100%–130%，步进 10%
- **响应式布局** — 适配桌面端、平板、手机三档断点
- **自定义滚动条** — 减少视觉干扰
- **纯静态，零依赖后端** — 可直接部署到任意静态托管平台

## 快速开始

### 方式一：直接打开

在浏览器中直接打开 `site/index.html` 即可预览。

### 方式二：本地服务器（推荐）

```bash
cd site
python -m http.server 8000
```

然后访问 `http://localhost:8000`

### 方式三：启动脚本

- Windows: 双击 `start.bat`
- Linux/macOS: 运行 `bash start.sh`

## 编辑简历

### 在线编辑器

点击右下角工具栏的 **✎** 按钮打开编辑器面板，支持：

- 编辑个人信息（姓名、岗位、电话、邮箱、GitHub、头像等）
- 增删改查各模块内容（教育背景、工作经历、项目经验、专业技能等）
- 新增/删除/排序/隐藏模块
- 调整页面偏好（字体大小、字体族、时间轴字段选择）
- 实时预览，编辑后点击 **保存** 即刻生效

### 导入数据

编辑器支持导入：
- **JSON** — 标准 JSON 格式简历数据
- **Markdown** — 解析结构化 Markdown 为简历数据（支持 `##` 标题分节、`**粗体**` 标记字段名）

### 导出数据

- 导出为 JSON 文件
- 导出为 Markdown 文件

### 重置数据

点击 **重置默认** 按钮恢复初始数据。

### URL 参数快速进入编辑

在 URL 后附加 `?edit=1` 打开页面时自动进入编辑模式。

## PDF 导出

点击编辑器底部 **导出 PDF** 按钮，或页面加载后使用 html2pdf.js 生成 PDF 文件。

## 页面模块

可通过编辑器灵活配置以下模块类型：

| 模块 | 说明 |
|------|------|
| 个人信息 | 姓名、岗位、联系方式、头像、GitHub |
| 工作经历 | 公司、职位、时间、工作描述、亮点 |
| 教育背景 | 学校、专业、学历、时间、主修课程 |
| 项目经验 | 项目名、时间、技术栈、描述、亮点 |
| 专业技能 | 技能名 + 详情（网格布局） |
| 自我评价 | 自由文本条目 |
| 时间轴 | 时间、标题、标签、描述、亮点 |
| 自由文本 | 纯文本段落 |

## 缩放控制

| 操作 | 说明 |
|------|------|
| `+` | 放大 10%（最大 130%） |
| `−` | 缩小 10%（最小 100%） |
| `100%` | 显示当前比例，点击重置为 100% |
| 缩放步进 | 10% |

缩放时保持当前阅读区域稳定，不会跳回顶部。

## 自定义样式

在 `site/styles.css` 的 `:root` 中可以修改：

- `--accent` — 强调色
- `--shadow` — 页面阴影
- `--radius` — 圆角大小
- `--font-family` — 字体组合
- 各 `--fs-*` 变量 — 字号层级
- 各 `--space-*` 变量 — 间距控制

## 部署

纯静态项目，可直接部署到：

- **GitHub Pages**
- **Vercel**
- **Netlify**
- **任意 Nginx / Caddy / Apache 静态目录**

部署时确保上传 `site/` 目录全部文件（含 `assets/` 和字体文件）。

## 技术支持

- 所有数据存储在浏览器 `localStorage` 中，刷新不丢失
- `data.json` 作为初始数据源；编辑后数据优先从 `localStorage` 读取
- 无 JavaScript 时仍可阅读基础简历内容（降级显示）
- 兼容 Chromium 系浏览器，基础适配移动端
