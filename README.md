# CV 静态简历网页

一个使用原生 `HTML/CSS/JS` 实现的静态简历网页项目，采用 A4 纸张视觉布局，适合在桌面浏览器中在线展示个人简历。

## 项目特点
- A4 纸张比例布局，视觉上接近正式简历
- 原生静态实现，结构简单，易于部署
- 右下角悬浮工具栏支持放大、缩小、重置
- 默认 `100%` 显示，最小不低于 `100%`，最大支持 `130%`
- 页面保持水平居中，缩放时尽量维持当前阅读位置
- 支持根据内容自动分页，超出首屏内容会自动续到后续页面
- 手机号、邮箱支持点击复制
- 针对移动端做了基础适配
- 使用自定义滚动条样式，减少视觉干扰

## 目录结构
```text
cv/
├─ docs/
│  └─ requirement.md
├─ site/
│  ├─ index.html
│  ├─ styles.css
│  └─ main.js
└─ README.md
```

## 文件说明
- `docs/requirement.md`：项目需求文档
- `site/index.html`：简历页面结构与示例内容
- `site/styles.css`：A4 布局、视觉样式、工具栏样式、滚动条样式
- `site/main.js`：缩放控制、比例状态、滚动位置保持、联系方式复制反馈逻辑

## 当前实现
### 页面模块
- 个人信息
- 教育背景
- 实习经历
- 专业技能
- 项目经历

### 当前交互
- 手机号点击复制
- 邮箱点击复制
- GitHub 主页跳转
- 根据内容自动分页

### 缩放交互
- `+`：放大简历
- `-`：缩小简历
- `100% / 110% / 120% / 130%`：显示当前比例，点击可重置

### 缩放规则
- 默认比例：`100%`
- 最小比例：`100%`
- 最大比例：`130%`
- 缩放步进：`10%`

## 使用方式
### 本地预览
直接在浏览器中打开 `site/index.html` 即可。

也可以在项目目录下启动任意静态服务器，例如：

```powershell
cd D:\code\cv\site
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 如何替换为你的真实简历
### 1. 修改基础信息
编辑 `site/index.html` 中以下内容：
- 姓名
- 求职岗位
- 年龄 / 性别
- 电话
- 邮箱
- GitHub / 个人主页

### 2. 修改简历模块内容
按模块替换：
- 教育背景
- 实习经历
- 专业技能
- 项目经历

### 3. 保持页面密度
建议优先精简文案，避免内容过长影响 A4 视觉布局与分页观感。

## 可调整项
### 样式调整
在 `site/styles.css` 中可以修改：
- 强调色：`--accent`
- 页面阴影：`--shadow`
- 圆角：`--radius`
- 字体与字号层级
- A4 视觉留白与模块间距

### 缩放调整
在 `site/main.js` 中可以修改：
- `MAX_SCALE`：最大缩放值
- `STEP`：缩放步进
- 按钮行为和滚动保留逻辑

## 部署方式
这是一个纯静态项目，可直接部署到：
- GitHub Pages
- Vercel
- Netlify
- 任意 Nginx 静态站点目录
